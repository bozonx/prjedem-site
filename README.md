# Project Edem site

## Run

Use node version: 18

- Install dependencies:

  ```bash
  yarn
  ```

- Run locally:

  ```bash
  yarn dev
  ```

- Build:

  ```bash
  yarn build
  ```

- Preview and admin

  ```bash
  yarn build && yarn preview
  yarn admin
  ```

  Go to `http://localhost:4173/admin`
